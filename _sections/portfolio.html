저의 프로젝트들입니다.

Clone Coding

패스트푸드 프랜차이즈 맘스터치의 웹사이트를 클론 코딩한 프로젝트입니다. 전체적인 구성은 BootStrap5를 토대로 하였으며 해당 웹사이트에 있는 여러 기능을 slider, offcanvas, carousel 등을 이용해 구현하였습니다. 이를 통해 HTML5/CSS3에 대한 이해를 한층 더 높일 수 있었고, BootStrap5를 더욱 유연하게 활용할 수 있게 되었습니다.

Board Project

웹 사이트에서 빠질 수 없는 게시판을 만든 프로젝트입니다. 로그인 후 해당 홈페이지로 이동하여 게시판에 직접 글을 올리고, 다른 유저들이 올린 글도 읽을 수 있으며, 이를 수정하고 삭제할 수 있도록 구현했습니다. 또한 Ajax를 통해 해당 페이지를 새로고침하지 않더라도 올린 댓글들을 최신순으로 볼 수 있게 하여 게시판의 기능을 더욱 잘 살렸습니다. 해당 프로젝트를 통해 Java 언어에 대한 이해도를 더욱 높였고 DB와 연결하는 과정으로 SQL query 언어도 잘 활용할 수 있게 되었습니다.

Snack War Project

단종된 과자들의 재출시를 장려하기위해 만든 사이트입니다. 회원제로 운영되며 메인 화면에서 과자 회사를 클릭하면 해당하는 회사의 단종된 과자들을 볼 수 있습니다. 유저가 재출시를 원하는 과자를 클릭해 '좋아요'를 남기면 DB에서 그 수치를 모아 일정 기준 이상에 도달하면 해당 회사에 재출시를 요구하는 청원서를 올릴 수 있도록 설정했습니다. 이 밖에도, 웹사이트에 게시판을 만들어 다른 단종 과자들을 추천받을 수 있도록 했고 매달 '이 달의 과자', '이달의 명언'을 올려 유저들 간의 소통을 가능케하였습니다. 이 프로젝트를 기획, 구현하며 다양한 프로그래밍 언어와 툴, 라이브러리와 프레임워크를 경험할 수 있었습니다.
